Protecting Your Privacy

We use anonymous analytics to understand how the Lux Client is used and to improve our services and stability. No personal or identifiable data is linked to your browsing sessions or launcher usage.

What data do we collect?
Below is a complete and transparent list of all data the launcher and our servers collect:

Launcher Information

  • Launcher Version (e.g., 1.6.7, 1.3.3)
  • Operating System (Windows, Linux, macOS)

Game Session Data

  • Mod Loader / Server Software (e.g., Fabric, NeoForge, Vanilla)
  • Minecraft Version played (e.g., 1.21.11)
  • Mode (Client or Server)
  • Game launches per day (anonymous counter)

Active Users (Live)

  • Number of active users in the launcher (anonymous count)
  • Number of users currently in-game (anonymous count)
  • Which instances are being played (anonymous count per instance)

Download Tracking

  • Downloaded Mods (name and download count)
  • Downloaded Resource Packs (name and download count)
  • Downloaded Shaders (name and download count)
  • Downloaded Modpacks (name and download count)

Modpack Codes

  • Generated share codes (code, name, mods, resource packs, shaders, version, loader)
  • Code usage count
  • Code creation & expiry timestamps
  • Owner UUID (if logged in via launcher) or IP (if used via website, for rate limiting only)

Codes are temporary and auto-deleted after 5 days (website) or 7 days (launcher account).

Opting Out

Note: Modpack code data is functional (not analytics) and is collected only when you actively create or use a modpack share code.
